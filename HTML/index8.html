MongoDB


MongoDB is a popular, open-source NoSQL database that uses a document-oriented data model. It stores data in flexible, JSON-like documents, making it well-suited for applications with evolving data structures. Unlike traditional relational databases, MongoDB doesn't use tables and rows; instead, it organizes data into collections of documents


ExpressJS


Express.js is a minimal and flexible Node.js web application framework designed for building robust web applications and APIs. It simplifies the development of server-side applications by providing a streamlined API for handling common web development tasks.
